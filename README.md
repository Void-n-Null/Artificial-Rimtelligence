# Artificial Rimtelligence

I came back across this after leaving it alone for a while. On and off for about a year, I had the idea of making a RimWorld mod that used an agentic chat loop to help run a colony. The main inspiration was [RimGPT](https://github.com/pardeike/RimGPT), particularly its agentic narration work, but I wanted to push the idea toward an AI that could participate in colony management rather than only comment on what was happening.

The plan grew into a fairly large multi-mod setup, with a shared core and separate pieces handling different parts of the colony. It never really panned out. What is here is some early scaffolding and OpenRouter authentication work, not a finished or usable mod.

I am leaving the repository up as a record of the idea and the small amount of implementation that came out of it.
