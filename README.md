# Artificial Rimtelligence

This was my second run at putting an AI agent inside RimWorld. The first was [ColonyCopilot](https://github.com/Void-n-Null/ColonyCopilot) in April 2024, a working prototype where a GPT-4 tool-calling loop observed a live colony and issued in-game actions, shelved mostly because of API costs at the time. The idea kept simmering. [RimGPT](https://github.com/pardeike/RimGPT) and its agentic narration work were another inspiration, but I wanted an AI that could participate in colony management rather than only comment on what was happening.

The plan grew into a fairly large multi-mod setup, with a shared core and separate pieces handling different parts of the colony. It never really panned out. What is here is some early scaffolding and OpenRouter authentication work, not a finished or usable mod.

I am leaving the repository up as a record of the idea and the small amount of implementation that came out of it.
